There are 3 files in this zip file:
1) The source data file: cacm.txt
2) The README file: README.txt
3) The Python code file: word_pair.py

Steps of running the code:
1) Make sure cacm.txt file is in the same folder with word_pair.py
2) In terminal, type "python word_pair.py". 
3) It will take 15-20 seconds to display the results. There are 3 parts
   of the results: 
   a. the top 10 pairs with the highest counts
   b. the top 10 pairs with the highest mutual information
   c. the top 5 words which have the highest mutual information 
      with word "programming"
